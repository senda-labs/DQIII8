Que es DQ

DQ es un asistente de inteligencia artificial que vive en tu ordenador. Le dices lo que necesitas — redactar un email, analizar datos, explicar un concepto, escribir codigo — y el lo hace por ti.

DQ no es un modelo de IA. Es un sistema que coordina varios modelos de IA (como ChatGPT, pero diferentes empresas) y elige automaticamente cual usar segun tu tarea. El objetivo: resolver cada tarea con el modelo mas barato posible.

El 70% de las tareas se resuelven gratis. Solo cuando la tarea es muy compleja (analisis financiero avanzado, arquitectura de sistemas, razonamiento profundo) DQ usa un modelo de pago — y te cuesta pocos centimos.

💡
Analogia: Imagina que tienes un asistente personal muy listo. Para tareas sencillas usa su propio cerebro (gratis). Para tareas muy especializadas llama a un experto (cuesta algo). DQ hace lo mismo, pero automaticamente.

Los tres niveles (tiers) de DQ

Nivel Quien responde Coste Para que sirve
Tier C — Local Tu propio ordenador (Ollama) $0 Codigo, debug, git, tareas simples
Tier B — Groq Servidores de Groq (nube) $0 Investigacion, redaccion, analisis
Tier A — Claude Anthropic (nube) ~0,01-0,05€ Finanzas, creatividad, arquitectura compleja

En modo automatico (el recomendado), DQ decide que nivel usar. Tu no tienes que pensar en ello.

Crear tu cuenta de GitHub

Si ya tienes cuenta de GitHub, salta a la Seccion 4.

  • 1
    Abre tu navegador Chrome, Firefox, Safari, Edge — cualquiera sirve.
  • 2
    Ve a github.com Veras la pagina de inicio de GitHub.
  • 3
    Haz click en "Sign Up" Esta en la esquina superior derecha (boton verde o negro).
  • 4
    Pon tu email, crea una contrasena y elige un username El username es tu nombre publico en GitHub (p.ej. juangarcia). Lo necesitaremos mas adelante para enviarte la invitacion.
  • 5
    Resuelve el captcha y verifica tu email GitHub te mandara un email con un codigo de 6 digitos. Ponlo donde te lo pide.
  • 6
    Envianos tu username de GitHub Escribe a Iker o Guillermo con tu username para que te mandemos la invitacion. Sin este paso no puedes descargar DQ.
🔔
Apunta tu username. Lo necesitaras varias veces en esta guia. Es lo que aparece en github.com/tu-username.

Aceptar la invitacion de beta

Cuando te enviemos la invitacion, GitHub te mandara un email automatico.

  • 1
    Revisa tu bandeja de entrada Busca un email de noreply@github.com con el asunto "You've been invited to collaborate on a private repository" o similar.
  • 2

  • 3

⚠
Si no ves el email: Busca en la carpeta de spam o correo no deseado. Si tampoco esta ahi, visita directamente github.com/notifications — la invitacion aparece ahi aunque no hayas recibido el email.
5

Crear tu Token de GitHub

🔑
Que es un token: Es como una contrasena especial que le permite a tu ordenador descargar el software privado de DQ de GitHub. No es tu contrasena normal de GitHub — es una clave distinta que puedes revocar en cualquier momento. Solo la usaras durante la instalacion.
  • 1
    Ve a github.com/settings/tokens Si te pide hacer login, hazlo con tu cuenta de GitHub.
  • 2
    Haz click en "Generate new token" → "Generate new token (classic)" (No el "Fine-grained" — usa el classic.)
  • 3
    En el campo "Note", escribe un nombre Por ejemplo: dq-beta. Es solo para que lo identifiques.
  • 4
    En "Expiration", elige "90 days" Esto hace que el token caduque automaticamente en 90 dias por seguridad.
  • 5
    Marca SOLO la casilla "repo" Es la primera del listado grande. Esto es todo lo que DQ necesita: poder descargar el repositorio. No marques nada mas.
  • 6
    Haz scroll hasta abajo y click en "Generate token" (Boton verde al final de la pagina.)
  • 7
    COPIA EL TOKEN AHORA Aparecera un token que empieza por ghp_. Pulsalo para copiarlo al portapapeles, y guardalo en algon sitio seguro (bloc de notas, gestor de contrasenas).
⚠
GitHub SOLO muestra el token una vez. Si cierras la pagina sin copiarlo, tendras que crear uno nuevo. El proceso es el mismo — no es un problema grave, pero mejor copiarlo ahora.

Instalar DQ — elige tu sistema

Las instrucciones varian segun tu ordenador. Elige la pestana correcta:

Abrir la Terminal en Mac

La Terminal es una ventana donde escribes instrucciones directamente a tu ordenador, como si le hablaras por texto. Es mas rapida que hacer click en menus para ciertas tareas.

  • 1
    Abre el Spotlight Pulsa Cmd ⌘ + Espacio. Aparecera una barra de busqueda.
  • 2
    Escribe "Terminal" y pulsa Enter Se abrira una ventana negra o blanca con texto. Eso es la Terminal.
  • 3
    Copia y pega este comando (uno a uno, Enter despues de cada uno)
    Terminal
    git clone https://github.com/senda-labs/DQIII8-premium.git

    Cuando te pida Username: escribe tu username de GitHub.
    Cuando te pida Password: pega tu token ghp_... (NO tu contrasena de GitHub — el token que creaste en la Seccion 5). Nota: al pegar la contrasena en la Terminal, no veras ningun texto — es normal.

  • 4
    Entra en la carpeta e instala
    Terminal
    cd DQIII8-premium
bash install.sh

    La instalacion descarga lo necesario. Tarda entre 5 y 15 minutos segun tu conexion. Veras texto pasando — es normal.

  • 5
    Cierra la Terminal y vuelve a abrirla Esto es importante — recarga la configuracion para que el comando dq funcione.
  • 6
    Ejecuta el configurador
    Terminal
    dq --setup

    Esto lanza el wizard de configuracion. Lo explicamos en detalle en la Seccion 7.

Windows requiere WSL2 — un mini-Linux dentro de Windows

DQ necesita un entorno Linux. WSL2 (Windows Subsystem for Linux) es la solucion oficial de Microsoft: instala un Ubuntu completo dentro de tu Windows, sin necesidad de reiniciar ni particionar el disco. Es gratuito y oficial.

Paso 1: Instalar WSL2 y Ubuntu

  • 1
    Abre PowerShell como Administrador Haz click derecho en el menu de inicio (esquina inferior izquierda) → "Windows PowerShell (Administrador)" o "Terminal (Administrador)". Haz click en "Si" cuando pregunte si quieres permitir cambios.
  • 2
    Ejecuta este comando
    PowerShell (Admin)
    wsl --install -d Ubuntu-24.04

    Este comando instala WSL2 y Ubuntu 24.04 automaticamente. Tarda varios minutos.

  • 3
    Reinicia el ordenador cuando te lo pida Despues de reiniciar, se abrira automaticamente una ventana de Ubuntu pidiendo que crees un usuario y contrasena para Linux. Ponle un nombre simple (ej: dq) y una contrasena que no olvides.
  • 4
    Ya tienes Ubuntu dentro de Windows Desde ahora, busca "Ubuntu" en el menu de inicio para abrir la terminal de Linux.

Paso 2: Instalar DQ dentro de Ubuntu

Con Ubuntu abierto, ejecuta exactamente los mismos comandos que en Mac:

Ubuntu (WSL2)
git clone https://github.com/senda-labs/DQIII8-premium.git
cd DQIII8-premium
bash install.sh

Cuando pida Username: tu username de GitHub.
Cuando pida Password: tu token ghp_... (no veras nada al escribir — es normal).

Al terminar, cierra y vuelve a abrir Ubuntu, y ejecuta:

Ubuntu
dq --setup
💡
Para abrir Ubuntu en el futuro: Busca "Ubuntu" en el menu de inicio. O desde el nuevo Windows Terminal (recomendado), haz click en la flecha del boton "+" y selecciona "Ubuntu".

Abrir la Terminal en Linux

En la mayoria de distribuciones Linux, puedes abrir la Terminal con el atajo de teclado Ctrl + Alt + T. Tambien puedes buscarla en el menu de aplicaciones.

Con la Terminal abierta, ejecuta:

Cuando pida Username: tu username de GitHub.
Cuando pida Password: tu token ghp_....

Opcion para ordenadores sin WSL2 o usuarios avanzados

VirtualBox es un programa que crea un "ordenador virtual" dentro de tu ordenador. Como tener otro PC dentro de tu PC, pero sin necesitar hardware extra. Es gratis.

⚠
Esta opcion requiere mas pasos y tu ordenador necesita al menos 8 GB de RAM para que tanto tu sistema como la maquina virtual funcionen bien. Si tienes Windows 10/11 de 64 bits, recomendamos WSL2 en lugar de esto.

Paso 1: Instalar VirtualBox

  • 1
    Descarga VirtualBox Ve a virtualbox.org/wiki/Downloads y descarga la version para tu sistema operativo.
  • 2
    Instala VirtualBox Abre el instalador descargado y sigue los pasos (siguiente, siguiente, instalar). Si pregunta sobre drivers de red, di que si.

Paso 2: Descargar Ubuntu

  • 3
    Descarga la imagen de Ubuntu 24.04 Ve a ubuntu.com/download/desktop y descarga Ubuntu 24.04 LTS (archivo .iso, unos 5 GB).

Paso 3: Crear la maquina virtual

  • 4
    Abre VirtualBox → "Nueva" Dale un nombre (ej: "Ubuntu DQ"), tipo Linux, version Ubuntu 64-bit.
  • 5
    Asigna 4 GB de RAM (4096 MB) y 2 CPUs si puedes.
  • 6
    Crea un disco virtual de 25 GB
  • 7
    En la configuracion de la VM → Almacenamiento, selecciona el archivo .iso de Ubuntu que descargaste como disco optico.
  • 8
    Inicia la VM e instala Ubuntu. Sigue el wizard de instalacion de Ubuntu (idioma, zona horaria, usuario). Al terminar, reinicia la VM.

Paso 4: Instalar DQ dentro de Ubuntu

Con Ubuntu funcionando en la VM, abre la Terminal (Ctrl+Alt+T) y ejecuta los mismos comandos que en la pestana Linux.

⌛
Que instala el script: Python, Ollama (el motor de IA local), el modelo qwen2.5-coder (segun tu RAM), SQLite para guardar historial, y configura el comando dq en tu terminal. Todo queda en tu ordenador — nada se sube a ninguna nube.

Configurar DQ — el Setup Wizard

Despues de instalar, ejecuta:

Terminal
dq --setup

El wizard te guia por 4 pasos. Puedes saltarte los opcionales con Enter.

Paso 1 de 4 — Groq API Key (muy recomendado, gratis)

💡
Que es Groq: Una empresa americana que ofrece modelos de IA de forma gratuita. Sus servidores son muy rapidos (responde en 1-2 segundos). DQ los usa para tareas de redaccion, investigacion y analisis que no necesitan el modelo mas caro.

Que es una API Key: Una contrasena especial que le permite a DQ usar los servidores de Groq en tu nombre. Tu cuenta de Groq es gratis — no necesitas tarjeta de credito.

Como conseguir tu Groq API Key:

  • 1
    Ve a console.groq.com/keys Si no tienes cuenta, creala — es gratis y no pide tarjeta.
  • 2
    Haz click en "Create API Key" Dale un nombre (ej: dq-beta).
  • 3
    Copia la key — empieza por gsk_. Guardala como el token de GitHub: solo se muestra una vez.
  • 4
    En el wizard, pega la key cuando te la pida

Paso 2 de 4 — Anthropic API Key (opcional, cuesta dinero)

💡
Que es Anthropic: La empresa fundada en 2021 que creo Claude, uno de los modelos de IA mas avanzados del mundo. Sus fundadores son ex-empleados de OpenAI (los que crearon ChatGPT).

Que es Claude: Un modelo de IA similar a ChatGPT pero de Anthropic. DQ lo usa para tareas que requieren razonamiento profundo: analisis financiero, arquitectura de sistemas, razonamiento legal, escritura creativa avanzada.

Cuanto cuesta: Cada tarea cuesta entre 0,01 y 0,05 euros. Con 5 euros tienes para 100-500 tareas complejas. El 70% de las tareas normales van por Groq (gratis).
⚠
Este paso es completamente opcional. Si no lo configuras, DQ funciona perfectamente con Groq (gratis) para la gran mayoria de tareas. Puedes anadirlo mas adelante con dq --set-anthropic tu_key.

Si quieres configurarlo:

  • 1
    Ve a console.anthropic.com
  • 2
    Crea cuenta y anade un metodo de pago
  • 3
    Pon un limite mensual de 5€ en la configuracion de facturacion. Asi no te llevas ninguna sorpresa.
  • 4
    Ve a API Keys → Create API Key. Empieza por sk-ant-.
  • 5
    Pegala en el wizard

Paso 3 de 4 — Telegram (opcional, notificaciones en movil)

Esto te permite recibir notificaciones en tu telefono cuando DQ termina una tarea larga, o controlar DQ desde Telegram. Es completamente opcional. Pulsa Enter para saltarlo.

Paso 4 de 4 — Tier por defecto

Elige como quieres que DQ enrute las tareas:

Opcion Que hace Recomendado si...
auto DQ decide el nivel segun la tarea La mayoria de usuarios — deja que DQ elija
groq-only Todo por Groq (gratis, rapido) No has configurado Anthropic o quieres coste $0
groq+ollama Groq para pensar, Ollama local para codigo Quieres privacidad total en el codigo
ollama-only Todo en tu ordenador Sin internet, maxima privacidad (mas lento)

Recomendacion: elige auto si tienes Groq configurado. Es el modo mas inteligente.

El Dashboard — tu interfaz principal

El Dashboard es una pagina web que se abre en tu navegador y te permite usar DQ sin tocar la terminal. Es como una aplicacion de chat, pero mucho mas potente.

Para abrirlo:

Terminal
dq --dashboard

Luego abre http://localhost:8080 en tu navegador.

ℹ
La terminal tiene que quedarse abierta mientras usas el dashboard. Es la que mantiene el servidor en marcha. Si la cierras, el dashboard dejara de funcionar.

Las tres pestanas del Dashboard

Pestana Para que sirve
Chat Escribe lo que necesitas y DQ responde. Igual que usar ChatGPT, pero con DQ. Puedes adjuntar archivos PDF, Excel, CSV, imagenes.
Metrics Estadisticas de tu uso: cuantas tareas has hecho, cuales han funcionado, cuanto has gastado, salud del sistema (puntuacion 0-100).
Settings Cambiar tus API keys, el tier por defecto, y ver el estado de cada nivel.

Los botones de tier en el Chat

Encima del cuadro de texto veras unos botones: Auto / Local / Groq / Claude.

Boton Que hace
Auto DQ elige el mejor modelo para tu tarea (recomendado)
Local Usa solo tu ordenador (sin internet, mas lento)
Groq Fuerza el uso de Groq (gratis, muy rapido)
Claude Fuerza Claude de Anthropic (el mas capaz, puede costar unos centimos)

Funciones avanzadas del chat

  • Adjuntar archivos: Haz click en el clip (📎) para subir PDFs, Excel, CSVs o imagenes. DQ los leeara y podra responder sobre su contenido.
  • Voz: Haz click en el microfono (🎤) para dictar tu mensaje.
  • Leer en voz alta: Haz click en el altavoz que aparece debajo de cada respuesta para que DQ la lea en voz alta.
  • Temas: El boton "Dark / Night / Light" en la barra superior cambia el tema visual del dashboard.

Comandos utiles

Todos los comandos se ejecutan en la Terminal (o en Ubuntu si estas en Windows).

Comando Que hace
dq "tu tarea aqui" Ejecutar una tarea directamente. DQ la clasifica y responde.
dq --dashboard Abrir el dashboard web en localhost:8080. Mantener la terminal abierta.
dq --chat Modo conversacion en la terminal. Como el dashboard pero en texto.
dq --setup Lanzar el configurador de API keys y tier. Puedes ejecutarlo varias veces.
dq --status Ver que tiers estan disponibles y cuales no.
dq --audit Ver la salud del sistema (puntuacion 0-100). No necesita internet.

dq --set-anthropic sk-ant-TU_KEY Guardar la Anthropic API Key directamente.
dq --help Ver todos los comandos disponibles.

Ejemplo: clasificar antes de ejecutar

Terminal
dq --classify "Calculate 99% VaR for a 10-asset portfolio with Monte Carlo"

Te dira algo como: tier=A domain=social_sciences/finance. Asi puedes ver exactamente que modelo usaria DQ y por que, antes de gastar nada.

Problemas comunes y soluciones

Esto pasa porque la terminal abierta no ha cargado la nueva configuracion. Solucion rapida: cierra la terminal y vuelve a abrirla. Si sigue sin funcionar, ejecuta:

source ~/.bashrc

Si estas en Mac, puede que uses zsh en lugar de bash:

source ~/.zshrc

Este aviso es completamente normal si no has configurado la key de Anthropic. DQ funciona perfectamente con Groq (gratis). El aviso solo te informa de que el Tier A (Claude) no esta disponible. Puedes ignorarlo.

Si quieres activar Claude, ejecuta:

dq --setup

Ollama es el motor de IA local. A veces no arranca automaticamente. En modo automatico (auto), DQ lo detectara y usara Groq en su lugar — no te quedas sin respuesta.

Para arrancar Ollama manualmente (en una terminal nueva, dejala abierta):

ollama serve

Si tienes poca RAM (menos de 4 GB libres), es normal que Ollama falle. Usa dq --set-tier groq-only para evitarlo.

Comprueba que dq --dashboard sigue corriendo en la terminal. Si la terminal se cerro, ejecutalo de nuevo y vuelve a intentarlo.

Asegurate de que la URL es exactamente http://localhost:8080 (con http, no https).

Si usas WSL2 en Windows, el dashboard debe estar accesible en localhost:8080 desde tu navegador de Windows igualmente.

Esto significa que GitHub no te deja acceder al repositorio privado. Causas posibles:

  • No has aceptado la invitacion — revisa la Seccion 4.
  • Usaste tu contrasena de GitHub en lugar del token ghp_ — el campo "Password" en la terminal requiere el token, no tu contrasena normal.
  • El token no tiene el scope "repo" — crea uno nuevo siguiendo la Seccion 5.
  • El token caduco — crea uno nuevo en github.com/settings/tokens.

La parte mas lenta es descargar el modelo de IA (qwen2.5-coder: ~4-8 GB). Con una conexion normal puede tardar 15-30 minutos. Si ves texto moviendose, esta funcionando — espera.

Si llevas mas de 1 hora sin ver cambios, pulsa Ctrl+C para cancelar y ejecuta bash install.sh --no-model para saltar la descarga del modelo. Podras descargarlo despues con ollama pull qwen2.5-coder:7b.

Escribe tu mensaje en espanol y DQ respondera en espanol. El sistema es bilingue nativo — detecta el idioma de tu mensaje automaticamente.

dq "Explica que es el riesgo de credito en 3 puntos"

Seguridad y privacidad

  • 🔒
    Tus API keys son tuyas y solo tuyas
    Se guardan en un archivo .env en tu ordenador. DQ no las envia a ningun servidor nuestro. Solo salen de tu ordenador cuando DQ hace una peticion directa a Groq o Anthropic para responder tu tarea.
  • 🔒
    Tu token de GitHub es de solo lectura
    El scope "repo" solo permite descargar codigo. No puede modificar nada en tu cuenta de GitHub, ni enviarte emails, ni acceder a otros repositorios. Puedes revocarlo en cualquier momento desde github.com/settings/tokens.
  • 🔒
    Tus conversaciones no se envian a Senda Labs
    Cuando usas Tier C (local), todo queda en tu ordenador. Cuando usas Tier B (Groq) o Tier A (Claude), tu mensaje va directamente a esos servicios — igual que si usaras su web directamente. DQ no hace de intermediario.
  • 🔒
    La telemetria esta desactivada por defecto
    DQ no envia estadisticas de uso a ningun lado. Si en el futuro quisieras activarla para ayudarnos a mejorar el producto, podras hacerlo con DQIII8_TELEMETRY=true en tu .env.
  • 🔒
    El dashboard en remoto tiene autenticacion
    Si abres el dashboard en una IP publica (dq --dashboard --host 0.0.0.0), DQ genera automaticamente un token de sesion. Sin ese token, nadie puede acceder. En local (localhost:8080), no necesitas autenticacion.
✓
Resumen: Tu contrasena de GitHub, tus API keys, y tus conversaciones son privadas. DQ esta disenado para que los datos viajen directamente entre tu ordenador y el servicio de IA que estes usando — sin pasar por nosotros.

# Scoring Formulas and Award Criteria

## Legal Basis
Arts. 145–149 LCSP. Award must be based on best price-quality ratio (MEAT — Most Economically Advantageous Tender).
Pure lowest-price award is prohibited except for standardised, commodity-type contracts.

## Two Categories of Award Criteria

### 1. Automatic Criteria (Criterios Evaluables Automáticamente)
Quantifiable by formula without value judgement. Scored after opening Sobre C (economic).
Examples: price, delivery time, warranty period, energy consumption.

### 2. Value Judgement Criteria (Criterios de Juicio de Valor)
Require qualitative assessment. Scored before opening price (Sobre B).
Examples: technical methodology, work plan, team quality, innovation.

```
Rule: if value judgement criteria weight > 50% → specialist technical committee required
Rule: if value judgement criteria weight > 0% → strict separation of envelopes B and C
```

## Price Scoring Formulas

### Proportional (most common)
```
Score = (Best offer / Candidate's offer) × Max points
```
Example: best price = 80,000€; candidate = 90,000€; max = 40 points
Score = (80,000 / 90,000) × 40 = 35.56 points

### Square root variant (reduces spread)
```
Score = √(Best offer / Candidate's offer) × Max points
```

### Linear with floor (protects against temerarious bids)
```
If offer ≥ budget: Score = 0
If offer ≤ budget × 0.5: Score = Max points
Else: interpolate linearly
```

## Temerarious (Abnormally Low) Bids (Ofertas Anormalmente Bajas)

### Standard identification formula (art. 149 LCSP)
When ≥ 5 offers:
```
Temerarious threshold = arithmetic mean of all offers - 10%
Any offer below threshold = presumptively temerarious
```
When < 5 offers: authority sets its own threshold in pliego (typically mean - 15-25%).

### Process when temerarious bid identified
1. Contracting authority sends written request for justification (10 business days to respond)
2. Mesa de Contratación evaluates justification
3. If justified (e.g., innovative process, lower overheads): bid accepted
4. If unjustified: bid rejected, garantía definitiva = 10% (vs. standard 5%)
5. Reasoned decision published in PLACSP

## Mandatory Social and Environmental Criteria (art. 202 LCSP)

### Special Execution Conditions (Condiciones Especiales de Ejecución)
At least one social, ethical, or environmental condition must be included in every contract.
Must appear in PCAP as contract obligation (not award criterion).

Examples:
- Social: minimum % disabled workers, insertion contracts, gender equality plan
- Environmental: ISO 14001 / EMAS certification, sustainable materials, carbon footprint reporting
- Ethical: fair trade supply chain, code of conduct compliance

### Environmental Award Criteria (Criterios Medioambientales)
Eco-labels (etiquetas ecológicas): authority may require compliance with specific eco-label criteria as technical specification (not brand-specific label itself).

### Labour Insertion Contracts (Contratos Reservados — art. 99 LCSP)
Authorities may reserve contracts for:
- Centros especiales de empleo (sheltered workshops): persons with disability ≥ 30%
- Empresas de inserción (social enterprises): socially excluded persons

## Weight Distribution Guidance (typical pliego practice)
| Criteria type | Typical weight range |
|--------------|---------------------|
| Price | 40–60% |
| Technical proposal / methodology | 20–40% |
| Delivery time / schedule | 5–15% |
| Warranty / maintenance | 5–10% |
| Social/environmental | 5–10% |
| Improvements (mejoras) | 0–10% |

Improvements (mejoras) must be pre-defined and quantifiable; open-ended improvements are invalid.

# Humanities & Arts

This centroid covers human culture, expression, thought, and creativity.
It includes literature, philosophy, history, languages, and creative arts.

## Agents
- Literature: narrative structure, character development, worldbuilding, genres
- Philosophy: ethics, epistemology, logic, metaphysics, political philosophy
- History: historical method, periods, research methodology, historiography
- Languages: linguistics, translation, grammar, language learning
- Creative Writing: fiction, poetry, screenwriting, copywriting, storytelling

## When this centroid activates
- User asks to write stories, novels, chapters, characters
- User asks about philosophical concepts, ethics, arguments
- User asks about historical events, periods, methods
- User asks to translate, write creatively, or compose text
- Keywords: novel, story, chapter, character, philosophy, ethics, history, write, narrative

## Henderson-Hasselbalch Application

```
pH = pKa + log([A⁻]/[HA])

Buffer capacity beta = 2.303 * C * Ka*[H⁺] / (Ka + [H⁺])²
  Maximum at pH = pKa
  Useful range: pKa ± 1 (roughly 9:1 ratio either direction)

For polyprotic acids:
  pH 1: between pKa1 and pKa2 → use pKa1 in H-H
  pH 2: between pKa2 and pKa3 → use pKa2 in H-H
```

## Solubility Rules (Aqueous, 25°C)

SOLUBLE: all Na⁺, K⁺, NH₄⁺, NO₃⁻, CH₃COO⁻ salts; most Cl⁻ (except Ag⁺, Pb²⁺, Hg₂²⁺); most SO₄²⁻ (except Ba²⁺, Pb²⁺, Ca²⁺)

INSOLUBLE: most CO₃²⁻, PO₄³⁻, OH⁻, S²⁻ (except alkali metals and NH₄⁺)

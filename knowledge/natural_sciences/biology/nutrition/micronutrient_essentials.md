# Micronutrient Essentials — RDA Reference

RDA values for adult males (M) / females (F) unless noted.
Source: NIH Office of Dietary Supplements / Institute of Medicine DRI.

## Fat-Soluble Vitamins
| Nutrient | RDA M/F | Function | UL |
|----------|---------|----------|----|
| Vitamin A | 900/700 mcg RAE | Vision, immune, cell growth | 3000 mcg |
| Vitamin D | 600 IU (15 mcg) both; 800 IU if >70 | Bone, immune, calcium absorption | 4000 IU |
| Vitamin E | 15 mg both | Antioxidant, immune | 1000 mg |
| Vitamin K | 120/90 mcg | Blood clotting, bone metabolism | None set |

## Water-Soluble Vitamins
| Nutrient | RDA M/F | Function |
|----------|---------|----------|
| Vitamin C | 90/75 mg | Antioxidant, collagen synthesis, iron absorption |
| B12 (cobalamin) | 2.4 mcg both | Nerve function, DNA synthesis, RBC formation |
| Folate | 400 mcg DFE | Cell division, DNA synthesis (800 mcg if pregnant) |
| B6 | 1.3–1.7/1.3–1.5 mg | Protein metabolism, neurotransmitters |
| B1 (thiamine) | 1.2/1.1 mg | Energy metabolism |

## Minerals
| Nutrient | RDA M/F | Notes |
|----------|---------|-------|
| Calcium | 1000 mg both; 1200 if >50F or >70M | Bone, muscle contraction |
| Iron | 8/18 mg | Oxygen transport; women need more pre-menopause; 8 mg post-menopause |
| Magnesium | 420/320 mg | Muscle, nerve, 300+ enzymatic reactions |
| Zinc | 11/8 mg | Immune, wound healing, testosterone |
| Potassium | 3400/2600 mg | Blood pressure, muscle/nerve function |
| Sodium | <2300 mg/day max | Hypertension risk above this level |
| Selenium | 55 mcg both | Antioxidant, thyroid function |
| Iodine | 150 mcg both | Thyroid hormone production |

## Essential Fatty Acids
```
Omega-3 (EPA+DHA): 250–500 mg/day (cardiovascular and brain health)
ALA (plant omega-3): 1.6/1.1 g/day (conversion to EPA/DHA is ~5–15%)
Omega-6 (linoleic acid): 17/12 g/day
```

## Deficiency Risk Groups
- Vitamin D: indoor workers, dark skin, >65yo
- B12: vegans, vegetarians, >50yo (reduced absorption)
- Iron: premenopausal women, endurance athletes
- Calcium: low dairy intake, high sodium diet
- Omega-3: low fish/seafood intake

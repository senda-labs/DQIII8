# Financial Ratio Analysis

## Profitability
```
Gross Margin = Gross Profit / Revenue
Operating Margin = EBIT / Revenue
Net Margin = Net Income / Revenue
ROE = Net Income / Equity
ROA = Net Income / Total Assets
ROIC = NOPAT / Invested Capital
```
DuPont: ROE = Margin × Turnover × Leverage

## Liquidity
```
Current Ratio = Current Assets / Current Liabilities (>1.5 healthy)
Quick Ratio = (CA - Inventory) / CL (>1.0 healthy)
Cash Ratio = Cash / CL
```

## Leverage
```
Debt/Equity = Total Debt / Equity
Debt/EBITDA = Total Debt / EBITDA (<3x investment grade)
Interest Coverage = EBIT / Interest Expense (>3x safe)
```

## Efficiency
```
Asset Turnover = Revenue / Total Assets
Inventory Days = (Inventory / COGS) × 365
Receivable Days = (AR / Revenue) × 365
Payable Days = (AP / COGS) × 365
Cash Conversion Cycle = Inv Days + Rec Days - Pay Days
```

## Valuation
```
P/E = Price / EPS (S&P 500 avg: 15-20x)
EV/EBITDA = Enterprise Value / EBITDA (varies by industry)
P/B = Price / Book Value per Share
PEG = (P/E) / EPS Growth Rate (< 1 = undervalued)
```
EV = Market Cap + Debt - Cash

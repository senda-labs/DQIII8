# Formal Sciences

This centroid covers abstract systems of knowledge based on formal logic,
mathematics, and computation. It deals with structures, proofs, and algorithms
rather than empirical observation.

## Agents
- Mathematics: algebra, calculus, number theory, geometry, topology, linear algebra
- Statistics: probability, hypothesis testing, regression, Bayesian inference, distributions
- Logic: propositional logic, predicate logic, formal proofs, set theory
- Algorithms: data structures, complexity theory, sorting, searching, graph theory

## When this centroid activates
- User asks to prove, calculate, compute, or derive something mathematical
- User needs statistical analysis, probability, or data modeling
- User asks about algorithms, complexity, or data structures
- Keywords: equation, theorem, proof, probability, matrix, integral, derivative, algorithm

# Algebra Fundamentals

## Definition
Algebra is the branch of mathematics dealing with symbols and the rules for manipulating those symbols to represent numbers and quantities in formulas and equations. It provides the foundation for expressing mathematical relationships abstractly and solving for unknown values.

## Core Concepts

- **Variables and Expressions:** Symbols (x, y, z) represent unknown or variable quantities. Expressions combine variables and constants using arithmetic operations.
- **Equations and Inequalities:** An equation asserts equality between two expressions (e.g., 2x + 3 = 7). Inequalities express ordering relationships (>, <, >=, <=).
- **Linear Algebra:** Study of linear equations, matrices, vectors, and linear transformations. A linear equation in n variables defines a hyperplane in n-dimensional space.
- **Polynomials:** Expressions of the form a_n*x^n + ... + a_1*x + a_0. Key operations: factoring, roots (zeros), polynomial division.
- **Systems of Equations:** Multiple equations with multiple unknowns. Solved via substitution, elimination, or matrix methods (Gaussian elimination, Cramer's rule).
- **Functions:** Mappings from one set to another. Key types: linear f(x) = mx + b, quadratic f(x) = ax^2 + bx + c, exponential, logarithmic.
- **Abstract Algebra:** Study of algebraic structures — groups, rings, fields. Fundamental to cryptography and coding theory.

## Key Formulas
- Quadratic formula: x = (-b ± sqrt(b^2 - 4ac)) / 2a
- Difference of squares: a^2 - b^2 = (a+b)(a-b)
- Sum of arithmetic series: S_n = n(a_1 + a_n) / 2
- Matrix inverse: A^-1 exists iff det(A) ≠ 0

## Practical Applications
- **Data science:** Linear algebra underpins machine learning — gradient descent, PCA, neural network weight matrices.
- **Engineering:** Solving systems of equations for circuit analysis, structural loads, fluid dynamics.
- **Cryptography:** Modular arithmetic and field theory form the basis of RSA and elliptic curve encryption.
- **Economics:** Input-output models, linear programming for optimization.
- **Computer graphics:** Matrix transformations for rotation, scaling, projection.

# LCSP Overview — Ley 9/2017 de Contratos del Sector Público

## Legal Framework
- Ley 9/2017, de 8 de noviembre (LCSP) — transposes EU Directives 2014/23/UE, 2014/24/UE, 2014/25/UE
- Applies to: all public sector entities (Administración General del Estado — AGE, CCAA, local entities, public bodies)
- Superseded TRLCSP (RDL 3/2011)
- Key authority: Junta Consultiva de Contratación Pública del Estado (JCCPE)

## 8 Contract Types (art. 12-17 LCSP)
| Type | Spanish term | Definition |
|------|-------------|------------|
| Works | Contrato de obras | Construction, major renovation, civil works |
| Services | Contrato de servicios | Professional services, studies, consulting, maintenance |
| Supplies | Contrato de suministros | Purchase, lease, or hire of goods/products |
| Works concession | Concesión de obras | Design+build+operate; revenue risk on contractor |
| Services concession | Concesión de servicios | Service delivery; operational risk on contractor |
| Mixed | Contratos mixtos | Combination of above (dominant element determines rules) |
| Special public law | Contratos adm. especiales | Atypical contracts with public interest object |
| Private | Contratos privados | Subject to civil/commercial law; public procurement for award only |

## SARA Thresholds 2026–2027 (EU-harmonized, below = national rules apply)
| Category | Threshold |
|----------|-----------|
| Works (obras) | 5,538,000 € |
| Services — AGE / entidades del sector público estatal | 143,000 € |
| Services — CCAA, local entities, other public bodies | 221,000 € |
| Services — local entities (municipios) below 216,000 € | 216,000 € |
| Supplies — AGE | 143,000 € |
| Supplies — CCAA / local | 221,000 € |
| Services concession / works concession | 5,538,000 € |

Contracts above SARA thresholds must be published in DOUE (Official Journal of the EU).

## Key Principles (art. 1 LCSP)
- **Transparencia**: all contract information publicly accessible (PLACSP)
- **Igualdad de trato y no discriminación**: identical conditions for all tenderers
- **Libre concurrencia**: no unjustified restrictions on access
- **Integridad**: anti-corruption measures, conflict of interest declaration
- **Proporcionalidad**: requirements proportional to contract object and value
- **Eficiencia en el uso de fondos públicos**: best value for money, not just lowest price

## Estimated Contract Value (Valor Estimado del Contrato — VEC)
```
VEC = contract duration value + optional extensions + all modifications
      (excluding VAT/IVA)
```
- VEC determines applicable procedure and thresholds
- For framework agreements: VEC = max total value of all call-offs
- Must be calculated before publishing — cannot be modified to avoid SARA rules

## Prohibited Splitting (fraccionamiento)
Artificially splitting contracts to fall below thresholds is explicitly prohibited (art. 99 LCSP). Sanctions: nullity of contract, disciplinary proceedings.
